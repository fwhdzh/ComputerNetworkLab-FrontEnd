# 计算机网络实验

使用JWT实现单点登陆并对microservices授权，客户端使用Flutter，服务器端对每一个Microservices的调用需能截取JWT，并根据JWT和存储的权限进行判断以确定是否有权使用指定的Microservices，用户标识和权限存放在Apache Ignite中。
